Web Scraping

Raspando dados

Introduo

O que

  • Tcnica de extrair dados de websites.
  • Tirar vantagem da estrutura das web pginas para extrair e armazenar dados em um formato para anlise de dados.

Por que usar

  • Formato dos Dados: Pegar dados de sites e colocar em formatos desejados prontos para anlise.
  • No precisa copiar e colar: mesmo se fizermos isso, o formato poder no estar pronto para anlise.
  • Sem salvar/baixar: Quando no h opo de salvar/baixar o dado.
  • Automao: possvel automatizar o processo de extrao de dados.

Como

  • robots.txt
  • acessar o link
  • extrair/ armazenar/ analizar

Casos de Uso

  • Comparao de preos;
  • Review de produtos;
  • Informaes de esportes;
  • Dados de media social.

Modos de Raspar Dados

  • Copiar e colar;
  • Programas especficos;
  • API: Interface de Programao de Aplicao (pt-BR)), cujo acrnimo API provm do Ingls Application Programming Interface. De modo geral, a API composta por uma srie de funes acessveis somente por programao, e que permitem utilizar caractersticas do software menos evidentes ao utilizador tradicional
  • Document Object Model (DOM).
